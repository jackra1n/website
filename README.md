# My personal website

My personal website to show what I'm doing.

## Tech

- SvelteKit with `@sveltejs/adapter-static`
- Svelte 5 and TypeScript
- Vite
- UnoCSS (Wind v4, Typography, Icons, Web Fonts)
- Iconify icons (Lucide, Simple Icons)

## Develop

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
